### Keycloak โครงสร้างใหญ่
| ชิ้นส่วน           | เปรียบเทียบกับบริษัท | หน้าที่                                    |
| ------------------ | -------------------- | ------------------------------------------ |
| **Realm**          | บริษัท 1 แห่ง        | ศูนย์รวม User / Roles / Policies / Clients |
| **Client**         | ระบบหรือแอปแต่ละตัว  | คนที่มาขอ Login / Token                    |
| **User**           | พนักงาน              | คนที่จะล็อกอินจริง                         |
| **Roles & Groups** | ตำแหน่ง + แผนก       | ใช้กำหนดสิทธิ์                             |

```bash
Keycloak
 └── Realm (บริษัท)
     ├── Users (พนักงาน/Account)
     ├── Groups (แผนก/โครงสร้างองค์กร)
     ├── Roles (สิทธิ์/หน้าที่)
     └── Clients (แอปที่ใช้ Login ผ่านระบบนี้)
```
